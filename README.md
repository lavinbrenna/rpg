# _Cyberpunk RPG_

#### By _**Brenna Lavin, Ryan Bass**_

#### _Create a character and fight some enemies_

## Technologies Used

* HTML
* CSS
* JavaScript
* jQuery
* BootStrap
* Jest
* Webpack
* ESLint
* Babel
* Popper.js

## Description

_This simple RPG was at first made to get familiar with testing with Jest while using object oriented programming, but is now a growing project which allows for the user to create their own character or for them to have it randomly generated. Currently the combat logic is in progress, but is on the way!_

## Setup/Installation Requirements

* Clone git repository to local machine
* Navigate to root folder using machine's command line or terminal window in VSCode
* type ```$ npm install``` to install project dependencies
* type ```$ npm run test``` to run developer tests
* type ```$ npm run start``` to start live server and open in browser.
